How It Works

Three steps. Fully autonomous.

1

Deposit USDC

Agent deposits USDC into a program-owned vault on Solana. Funds are held in escrow.

2

Make API Calls

Agent sends requests through the Agent Pay proxy to any LLM provider ‚Äî OpenAI, Anthropic, and more.

3

Auto-Deduct

Usage is metered and costs are automatically deducted from the agent's USDC balance on-chain.

Developer Experience

Simple integration

// 1. Deposit USDC to your agent vault
const vault = await agentPay.createVault({
  agent: agentKeypair.publicKey,
  depositAmount: usdcToLamports(100),  // $100 USDC
});

// 2. Make API calls through the proxy ‚Äî usage auto-deducted
const response = await fetch("https://proxy.agentpay.sol/v1/chat/completions", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${vault.apiKey}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    model: "gpt-4",
    messages: [{ role: "user", content: "Hello from an autonomous agent!" }],
  }),
});

// 3. Check remaining balance
const balance = await agentPay.getBalance(vault.address);
console.log(`Remaining: $${balance} USDC`);
