How It Works

Three steps. Fully autonomous.

1

Deposit USDC

Agent deposits USDC into a program-owned vault on Solana. Funds held in escrow.

2

Make API Calls

Agent sends requests through Agent Pay proxy to any LLM provider.

3

Auto-Deduct

Usage metered, costs auto-deducted from agent's USDC balance on-chain.

Developer Experience

Simple integration

// 1. Deposit USDC to your agent vault
const vault = await agentPay.createVault({
  agent: agentKeypair.publicKey,
  depositAmount: usdcToLamports(100),  // $100 USDC
});

// 2. Make API calls ‚Äî usage auto-deducted
const response = await fetch("https://proxy.agentpay.sol/v1/chat/completions", {
  headers: { "Authorization": `Bearer ${vault.apiKey}` },
  method: "POST",
  body: JSON.stringify({
    model: "gpt-4",
    messages: [{ role: "user", content: "Hello from an autonomous agent!" }],
  }),
});

// 3. Check balance
const balance = await agentPay.getBalance(vault.address);
console.log(`Remaining: $${balance} USDC`);
