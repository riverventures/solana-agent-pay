Solution

Three Steps. Fully Autonomous.

Solana Agent Pay is a payment proxy that converts USDC deposits into API access. Agents deposit once, then make calls forever ‚Äî usage is metered and deducted automatically.

Deposit USDC to Vault

Each agent gets a unique vault address (PDA on Solana). Send USDC via any wallet ‚Äî Phantom, CLI, or programmatically. Funds are held in a non-custodial smart contract.

Make API Calls via Proxy

Point your agent's API calls through Agent Pay's proxy endpoint. It authenticates with the upstream provider (OpenAI, Anthropic, etc.) and forwards requests transparently.

Usage Auto-Deducted

Each API call is metered by token count and model tier. Costs are deducted from the agent's USDC balance in real-time. On-chain receipts for every transaction.

Developer Experience

Drop-in API Compatibility

Change one line ‚Äî your base URL. Everything else stays the same. Works with any OpenAI-compatible SDK.
